# Как получить APK — пошаговая инструкция

## Что нужно сделать (один раз)

### Шаг 1 — Создать аккаунт на GitHub
1. Откройте в браузере: **github.com**
2. Нажмите **Sign up** и зарегистрируйтесь (email + пароль)
3. Подтвердите email

### Шаг 2 — Создать репозиторий на GitHub
1. На главной странице GitHub нажмите **"+"** → **New repository**
2. Название: `ar-ruler-app`
3. Выберите **Public** (бесплатная сборка)
4. Нажмите **Create repository**
5. Скопируйте ссылку репозитория (вида `https://github.com/ВашеИмя/ar-ruler-app.git`)

### Шаг 3 — Подключить Replit к GitHub
1. В Replit нажмите **Git** (иконка ветки слева)
2. Нажмите **Connect to GitHub**
3. Войдите в свой GitHub аккаунт
4. Выберите репозиторий `ar-ruler-app`
5. Нажмите **Push** (отправить код на GitHub)

### Шаг 4 — Подождать сборку APK (~5-10 минут)
1. Откройте ваш репозиторий на GitHub
2. Нажмите вкладку **Actions** (вверху страницы)
3. Вы увидите процесс сборки "Build AR Ruler APK"
4. Подождите пока появится зелёная галочка ✅
5. Нажмите на задание → **ar-ruler-release-apk** → скачайте архив с APK

### Шаг 5 — Установить APK на телефон
1. Распакуйте архив — внутри файл `app-release.apk`
2. Перекиньте APK на телефон (через telegram/почту/кабель)
3. Откройте файл на телефоне → **Установить**
   - Если спросит про "неизвестные источники" — разрешите

## Когда захотите обновить приложение

Просто:
1. Измените код в Replit
2. Нажмите **Push** в Git панели
3. GitHub автоматически пересоберёт APK

## Что делает это приложение (Этапы 0-2)

- **Список проектов** — создавайте и управляйте проектами
- **AR Сканирование** — тапайте по реальным поверхностям, ставьте точки, получайте расстояние
  - Режим Бокс: 4 точки по углам → сохраняет параллелепипед с размерами
  - Режим Цилиндр: 2 точки по краям диаметра → сохраняет цилиндр
- **3D Просмотр** — интерактивный 3D-вид всех отсканированных объектов (вращение, масштаб пальцами)

Следующие этапы (3-5) добавят: сложные полигоны, экспорт в OBJ, синхронизацию AR-позиций.
